# Phase 4: 检索增强生成 (RAG)

## 07_advanced_rag

**目标**: 应对长文档分析与复杂的检索查询，构建可用于生产环境的基础 RAG。

**学习内容**:
- 引入向量数据库引擎（如 Chroma 或 FAISS）。
- 实践高级文档分块策略（Text Splitters）。
- 实现 LangChain Retriever，重点实践**混合检索（Hybrid Search: BM25 关键词检索 + 向量检索）**。
- 探索多路召回与重排序（Reranking）等高级 RAG 策略。

**技术栈**:
- Python
- LangChain / VectorStores (ChromaDB)
