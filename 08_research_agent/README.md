# Phase 5: 规划与复杂任务执行 (Planning)

## 08_research_agent

**目标**: 理解并应用经典的 ReAct 模式，让 Agent 具备思考、观察并执行长线任务的能力。

**学习内容**:
- 深入基于 LangChain AgentExecutor 的 ReAct (Reason + Act) 工作流。
- 构建一个能主动搜集信息的研究 Agent。
- Agent 在缺乏信息时会自主调用搜索引擎进行查找。
- 最终能将搜集到的资料汇总并输出结构化长篇报告。

**技术栈**:
- Python
- LangChain Agents
- Search Tools (Tavily / DuckDuckGo 等)
