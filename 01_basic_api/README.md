# Phase 1: 基础理论与大模型交互 (Basics & API)

## 01_basic_api

**目标**: 学习如何调用大模型的 API（兼容 OpenAI / Anthropic 等），并引入 LangChain 基础核心概念。

**学习内容**:
- 直接调用大模型 API 进行简单的文本生成。
- 掌握 **Prompt Template** 构造上下文。
- 掌握 **Output Parser** 按特定格式解析模型输出内容。
- 体验并实现流式响应（Streaming）。

**技术栈**:
- Python
- LangChain / LangChain-Core
