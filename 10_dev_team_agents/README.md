# Phase 6: 多智能体协同 (Multi-Agent Systems)

**前置依赖**: Phase 5

## 10_dev_team_agents (原 09_dev_team_agents)

**目标**: 探索并实践多个专注不同职责的 Agent 如何在系统中协同工作。

**学习内容**:
- 定义多个不同角色和技能的 Node/Agent (产品经理、程序员、测试员)。
- 设计它们之间通过状态图（Graph）交替传递信息的过程。
- 最终输出一段能够执行的代码或功能模块。

**技术栈**:
- Python (3.10+)
- LangGraph (`langgraph >= 0.1.0`)
